# Jumping Frog

**Simple console video game using ncurses library.**

Full instruction with requirements and goals for this project can be found [here](/docs/instruction-EN%20(early%20version%200.94).pdf).

## Building

Due to my personal aversion to _conio.h_ library and thus the usage of _ncurses_, MSYS2 is recommended to build this project on Windows.
